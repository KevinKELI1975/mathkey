Texting Manual

T.1 Backup binary file with extension ".bak":

The File > Open will save current files as a binary backup before the file content presents, the backup file has extension "bak".
To open the bak file, do it by "File > Open Binary".

T.2 Setup page border:

On the canvas, dashed lines are displayed to represent page borders. You can select A3/A4/A5 with portrait or landscape layout. Choose page sizes in "Setup > Page Size" menu.


  • Do "Setup > Add Page Row" to increase the page row layout.
  • Do "Setup > Add Page Column" to increase the page column layout.

T.3 Switching between text and math:

Use "t" / "m" button to switch between text / math typing.

T.4 High-light the empty line:

Show/Hide an empty line for visibility, by "Text > Highlight Empty".

T.5 Select a section of text:

  1. Press and keep Shift key.
  2. Click the mouse left key on the first location.
  3. Keep press the mouse left key and drag the mouse.

Select the whole line by double-click the mouse left button.


With a selected text, "Ctrl + X, Ctrl + C" and "Ctrl + V" are used for cut, copy and paste the selected text.

T.6 Paste text copied from other text editor:

In normal text editor, a copied section of text is stored in the clipboard, the text can be pasted into Mathkey editor by "Edit > Copy Clipboard".

Drawing Manual

D.1 Enable grids: Draw > Catch grid point

When catch grid point is enabled, mouse will capture on grid point when left button is clicked.

D.2 Snap on vertices of a drawing: Draw > Catch shape vertex

When catch shape vertex is enabled, mouse will capture on vertex point when left button is clicked.

D.3 Draw polygon, polyline, spline and Bezier curve:

In "Draw > polygon / polyline / Bezier / Spline" to draw the poly shapes.
The maximum number of vertices of a Bezier curve is 7 points.

D.4 Find intercept on item:

Find interception of polyline on a shape.

D.5 Create a mirrored item:

Create a mirrored item with respect to a straight line.

D.6 Circular array:

Make a circular array of an item around a vertex point, with requried angle and number.

D.7 Paste an image snapshot:

Here is how to paste an image snapshot into Mathkey.
- Step 1: snapshot a part of screen with windows snipshot tool.
- Step 2: "Ctrl + C" to paste the snapshot image.

D.8 Text and Label objects:

You can create text and label objects from drawing drag and drop list.
Here is the difference between these 2 objects.
- Text: a multi-line text section that allows you to put math symbols. A text object cannot be rotated.
- Label: characters only, single line object to note axes or graphics. A label object can be rotated.

D.9 Dimensioning:

Dimensioning object can be created by "Draw > Dimension Label". When a dimension label is first time
created, the length will be computed by on-paper length * Dimension Factor, afterwards, it can only be
changed manually. Dimension Factor can be changed by "Draw > Dimension Factor" in the menu.

Here are how you can change the length label and extension.
- Select the dimensioning object. - Length text: on the mouse right-button menu, click on "set string".

- Extension: on the mouse right-button menu, click on "set extension".

D.10 Layer management:

You can create / delete layer for graphics, do layer management in "Setup > Layers".

Several rules in layer manager:

- You must have one working layer and only one.
- Working layer cannot be deleted.
- You can only set current working layer non-working layer by setting other layer as working layer.
- A layer can only be deleted if it is not working layer and not occupied by any graphics.


D.11 Graphics Library:

At left-hand side of application screen, there are upper and lower windows to display "library" folder archive.
Application visits library folder and list graphics objects that matches specific format.

D.12 Create library objects:

You can save a selected graphics or a graphics group into the library, creating a graphics item to reproduce the graphics with drag and drop operation.

Procedure:
1. Select the library folder that you want to put an item.
2. Select the group item or a mono shape you would like to archive.
3. Click on righ mouse button and click on "into library...".

4. Create a bmp icon follow the bmp file name in the txg file, the prefered icon size is 32 x 32 pixels.
