Texting Manual

T.1 Switching between text and math:

Mathkey by default uses italic font as math font and non-italic font as text font. You can switch these two modes by "Text > text" or "Text > math", or t/m buttons.

T.2 Design your own math symbol style:

Mathematic symbols are store in the "func_symbol" folder, you can create your own artistic work by decoration on graphics.

T.3 Setup page border:

On the canvas, dashed lines are displayed to represent page borders. You can select A3/A4/A5 with portrait or landscape layout. Select borders in "Setup > Page Size" menu.


  • You can increase the page in row by "Setup > Add Page Row".
  • You can increase the page in column by "Setup > Add Page Column".

T.4 Open backup binary ".bak" file:

Mathkey will save your data in text format, it also automatic make a backup in binary file with file extension ".bak". In case you need to open the binary, do it in menu "File > open binary".

T.5 Select a section of text:

  1. Press shift key and keep it down.
  2. Click the mouse left key on the first position.
  3. Keep press the mouse left key and drag the mouse.


With a selected text, "Ctrl + X, Ctrl + C" and "Ctrl + V" are used for cut, copy and paste the selected text.

T.6 Paste text from clipboard:

When a section of text is copied and stored in clipboard, the text can be pasted into Mathkey editor with "Edit > Copy Clipboard".

T.7 High light the empty line:

You can high light an empty line as auxiliary input, do it in "Text > Highlight Empty".

Drawing Manual

D.1 Enable grids:

When grid is enabled, mouse will snap on grid point when you click on mouse left button.

D.2 Snap on vertices of a drawing:

When snapping on vertex is enabled, mouse will snap on vertex point when you click on mouse left button.

D.3 Draw polygon, polyline, spline and Bezier curve:

In "Draw >" menu, select polygon / polyline / Bezier / Spline to draw the poly shapes.
The maximum number of vertices of a Bezier curve is 7 vertices.

D.4 Paste a image snapshot:

Here is how to paste a image snapshot into Mathkey.
- Step 1: snapshot a part of screen with windows snapshot tool.
- Step 2: "Ctrl + C" to paste the snapshot image.

D.5 Text and Label objects:

You can create text and label objects from drawing drag and drop list.
Here is the difference between these 2 objects.
- Text: a multi-line text section that allows you to put math symbols. A text object cannot be rotated.
- Label: a character only, single line object to note axes or graphics. A label object can be rotated.

D.6 Dimensioning:

Dimensioning object can be created by "Draw > Dimension Label". When a dimension label is first time
created, the length will be computed by on-paper length * Dimension Factor, afterwards, it can only be
changed manually. Dimension Factor can be changed by "Draw > Dimension Factor" in the menu.

Here are how you can change the length label and extension.
- Select the dimensioning object. - Length text: on the mouse right-button menu, click on "set string".

- Extension: on the mouse right-button menu, click on "set extension".

D.6 Layer management:

You can create / delete layers for graphics, do layer management in "Setup > Layers".
- You must have one working layer and only one working layer.
- Working layer cannot be deleted.
- You can only set current working layer not working layer by setting other layer as working layer.
- A layer can only be deleted if it is not working layer and not occupied by any graphics.

D.7 Graphics Library:

At left-hand side of Mathkey there are upper and lower windows to display "library" folder archive.
The upper window displays folder structure that you can create to classify graphics.
The lower window displays graphics stored in the selected folder

D.8 Create dwawing element:

You can save a selected graphics or a graphics group into the library, create a graphics item to allow you for drag and drop operation.
Below are the steps to follow:
1. Select the library folder that you want to put a item.
2. Select the group item or a mono item you would like to put in the library.
3. Click on righ mouse button and click on "into library...".

4. Create a bmp icon follow the bmp file name in the txg file, the prefered icon size is 32 x 32 pixels.
